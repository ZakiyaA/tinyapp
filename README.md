## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.

# Getting Started
- Install all dependencies (using the npm install command).
- Run the development web server using the node express_server.js command.
- Open http://localhost:8080/urls/ in your browser
